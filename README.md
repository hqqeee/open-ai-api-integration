# OpenAI WebSocket App

A simple WebSocket server that communicates with the OpenAI API.


## Set-up
* Create venv
```shell
python -m venv venv
```
* Activate venv
Linux: 
```shell
source venv/bin/activate
```
Windows: 
```shell
venv\Scripts\activate
```
* Install required packages
```shell
pip install -r requirements.txt
```
